# Setup lab
1. Instalar XAMPP
2. Descargar Zip https://github.com/digininja/DVWA
3. Mover el contenido del Zip a la carpeta del servidor Xampp.
4. Activar Apache y Mysql en Xampp
5. Renombrar el config/config.inc.php.dist a config/config.inc.php
6. Crear un usuario en la phpmyadmin para configurarlo en el config/config.inc.php.
7. Entrar a localhost para instalar el app.
8. Hacer login en el app. Default credentials: admin / password
9. Corregir los errores mostrados en el dashboard.
10. Instalar Burp suite community.
